# Software Development Processes Powered by AI Agents

This repository contains coursework for a master's-level Software Development Process course, where the goal is to build AI agents that support and automate different stages of the development lifecycle.

Throughout the course, we focus on designing agents with well-defined inputs, rules, and constraints, and combining them with validation mechanisms such as conventions, hooks, and testing to ensure reliable behavior.

Each module introduces a specific development process and requires implementing it as an agent, which is then used as part of a larger system.
